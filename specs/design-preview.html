Color Balance

Minimal, not over-colored.

The site should lean heavily on light neutrals, with charcoal for text and deep green for structure. Sage and coral stay secondary so the interface feels calm rather than loud.

Typography

Editorial warmth with clean UI readability.

The serif carries the premium mood. The sans keeps pricing, booking steps, and dashboards practical. This is the balance the visual system should preserve.

Heading system
Beauty booking with real calm and real clarity.
Cormorant-style display typography creates the luxury cue, but the page still feels grounded because the supporting text remains direct and modern.
Body and utility system

Manrope-style UI text works well for navigation, labels, pricing chips, booking steps, and admin modules because it feels current without drifting into generic SaaS.
